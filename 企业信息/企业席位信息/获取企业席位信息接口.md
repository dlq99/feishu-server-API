## 获取企业席位信息

获取租户下待分配的席位列表，包含席位名称、席位ID、数量及对应有效期。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/tenant/v2/tenant/assign_info_list/query` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 查询租户下的席位信息 |
| 权限要求 | tenant:tenant.product_assign_info:read |
