## 创建知识空间

此接口用于创建知识空间

💡 
 此接口不支持tenant access token（应用身份访问）

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/wiki/v2/spaces` |
| HTTP Method | POST |
| 支持的访问令牌 | user_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 创建、更新知识空间 <br> 查看、编辑和管理知识库 |
| 权限要求 | wiki:space:write_only <br> wiki:wiki |

### 请求体

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| name | string | 否 | 知识空间名称 <br> **示例**: 知识空间  |
| description | string | 否 | 知识空间描述 <br> **示例**: 知识空间描述  |
| open_sharing | string | 否 | 表示知识空间的分享状态 <br> **示例**: open <br> **数据校验规则**:<br>最大长度: 99最小长度: 0<br> **可选值**:<br>- open: 打开 <br>- closed: 关闭 <br> |

**请求体示例**:

```json
{
    "name": "知识空间",
    "description": "知识空间描述"
}
```

### 响应

**响应示例**:

```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "space": {
            "name": "知识空间",
            "description": "知识空间描述",
            "space_id": "1565676577122621"
        }
    }
}
```
