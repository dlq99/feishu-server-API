## 下载文件

下载云空间中的文件，如 PDF 文件。不包含飞书文档、电子表格以及多维表格等在线文档。该接口支持通过在请求头添加 `Range` 参数分片下载部分文件。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/drive/v1/files/:file_token/download` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token, user_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 上传、下载文件到云空间 <br> 下载云空间下的文件 <br> 查看和下载云空间中的文件 <br> 查看、评论、编辑和管理云空间中所有文件 <br> 查看、评论和下载云空间中所有文件 |
| 权限要求 | drive:drive <br> drive:drive:readonly <br> drive:file <br> drive:file:download <br> drive:file:readonly |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| file_token | string | 文件的 token，获取方式见[文件概述](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/reference/drive-v1/file/file-overview)。 |

        .app_secret("YOUR_APP_SECRET") \
        .log_level(lark.LogLevel.DEBUG) \
        .build()

    f = open(f"/file_path/{response.file_name}", "wb")
    f.write(response.file.read())
    f.close()


if __name__ == "__main__":
    main()
