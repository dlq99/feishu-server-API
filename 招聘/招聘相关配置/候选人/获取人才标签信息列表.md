## 获取人才标签信息列表

可通过关键词、ID 列表、标签类型、停用状态等获取人才标签信息列表，结果按照创建时间倒序排序。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/hire/v1/talent_tags` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 更新人才标签 <br> 获取人才标签 |
| 权限要求 | hire:talent_tag <br> hire:talent_tag:readonly |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| keyword | string | 否 | 搜索关键词 |
| id_list | array | 否 | 标签 ID 列表 |
| type | integer | 否 | 标签类型 |
| include_inactive | boolean | 否 | 是否包含停用标签 |
| page_size | integer | 否 | 分页大小 |
| page_token | string | 否 | 分页标记，第一次请求不填，表示从头开始遍历；分页查询结果还有更多项时会同时返回新的 page_token，下次遍历可采用该 page_token 获取查询结果 |
