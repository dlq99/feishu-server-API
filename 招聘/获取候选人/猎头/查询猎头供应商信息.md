## 查询猎头供应商信息

根据猎头供应商名称查询有合作关系的猎头供应商信息，包含猎头供应商ID、名称、联系人等。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/hire/v1/agencies/query` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 更新招聘猎头信息 <br> 获取用户 user ID <br> 获取招聘猎头信息 |
| 权限要求 | hire:agency:readonly <br> hire:agency |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| name | string | 是 | 猎头供应商名称，精准匹配查询(区分大小写) |
| user_id_type | string | 否 | 用户 ID 类型 |
### 响应

**响应示例**:

```json
{
    "code": 0,
    "msg": "ok",
    "data": {
        "items": [
            {
                "id": "6898173495386147079",

                "contactor_name": {
                    "zh_cn": "张三",
                    "en_us": "zhangsan"
                }
            }
        ]
    }
}
```

### 错误码

| HTTP状态码 | 错误码 | 描述 | 排查建议 |
| ---------- | ------ | ---- | -------- |
| 500 | 1002001 | 系统错误 | 请根据实际报错信息定位或咨询[技术支持](https://applink.feishu.cn/TLJpeNdW) |
| 400 | 1002002 | 参数错误 | 检查参数是否正确，例如类型，大小 |
| 400 | 1002409 | 合作关系不存在 | 请检查入参 `name` 是否正确 |
