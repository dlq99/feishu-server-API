## 创建个人信息

创建人员的个人信息。

❌ 
 接口已升级，推荐使用：[创建个人信息](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/corehr-v2/person/create)

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/corehr/v1/persons` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom |
| 权限要求 | 更新核心人事信息 <br> 读写员工的个人信息 |
| 权限要求 | corehr:corehr <br> corehr:person:write |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| client_token | string | 否 | 根据client_token是否一致来判断是否为同一请求 |
