## 获取单个用户信息

调用该接口获取通讯录中某一用户的信息，包括用户 ID、名称、邮箱、手机号、状态以及所属部门等信息。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/contact/v3/users/:user_id` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token, user_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 查询用户职级 <br> 获取成员所在部门路径 <br> 查看成员数据驻留地 <br> 查询用户席位信息 <br> 获取用户基本信息 <br> 查看成员工号 <br> 读取通讯录 <br> 以应用身份访问通讯录 <br> 以应用身份读取通讯录 <br> 获取通讯录基本信息 <br> 获取用户性别 <br> 获取用户邮箱信息 <br> 获取用户受雇信息 <br> 获取用户手机号 <br> 查询用户所属的工作序列 <br> 查看成员的虚线上级 ID <br> 获取用户组织架构信息 <br> 获取用户 user ID |
| 权限要求 | contact:contact.base:readonly <br> contact:contact:access_as_app <br> contact:contact:readonly <br> contact:contact:readonly_as_app |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| user_id | string | 用户ID。ID 类型与查询参数 `user_id_type` 保持一致。 |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| user_id_type | string | 否 | 用户 ID 类型 |
| department_id_type | string | 否 | 指定查询结果中的部门 ID 类型。关于部门 ID 的详细介绍，可参见[部门 ID 说明](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/reference/contact-v3/department/field-overview#23857fe0)。 |
