## 创建用户

调用该接口向通讯录创建一个用户（该动作可以理解为员工入职）。成功创建用户后，系统会以短信或邮件的形式向用户发送邀请，用户在同意邀请后方可访问企业或团队。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/contact/v3/users` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom |
| 权限要求 | 查看成员工号 <br> 查看成员数据驻留地 <br> 以应用身份访问通讯录 <br> 查询用户职级 <br> 获取用户性别 <br> 查询用户所属的工作序列 <br> 更新通讯录 <br> 获取用户手机号 <br> 获取用户组织架构信息 <br> 查看成员的虚线上级 ID <br> 获取用户基本信息 <br> 读取通讯录 <br> 以应用身份读取通讯录 <br> 获取用户 user ID <br> 获取用户受雇信息 <br> 获取用户邮箱信息 |
| 权限要求 | contact:contact |
