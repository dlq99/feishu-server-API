## 获取租户职级列表

调用该接口获取当前租户下的职级信息，包括职级名称、描述、排序、状态以及多语言等。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/contact/v3/job_levels` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 创建、删除、更新职级 <br> 以应用身份读取通讯录 <br> 查询职级列表 |
| 权限要求 | contact:job_level:readonly <br> contact:job_level <br> contact:contact:readonly_as_app |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| page_size | integer | 否 | 分页大小，用于限制一次请求所返回的数据条目数。 |
| page_token | string | 否 | 分页标记，第一次请求不填，表示从头开始遍历；分页查询结果还有更多项时会同时返回新的 page_token，下次遍历可采用该 page_token 获取查询结果 |
| name | string | 否 | 职级名称。

- 传入该字段时，可查询指定职级名称对应的职级信息（不支持模糊查询）。
- 不传入该字段时，查询当前租户下所以职级的信息。 |
