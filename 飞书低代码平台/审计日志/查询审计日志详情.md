## 飞书低代码平台-查询审计日志详情

根据日志 ID 查询审计日志详情

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/apaas/v1/applications/:namespace/audit_log` |
| HTTP Method | GET |
| 支持的访问令牌 | user_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 获取飞书低代码平台审计日志信息 |
| 权限要求 | app_engine:security.audit_log:read |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| namespace | string | 应用命名空间 |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| log_id | string | 是 | 审计日志ID信息（通过[查询审计日志列表](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/apaas-v1/application-audit_log/audit_log_list)获取单条日志ID） |
