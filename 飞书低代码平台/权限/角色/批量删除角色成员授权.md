## 批量删除角色成员授权

批量删除角色成员授权

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/apaas/v1/applications/:namespace/roles/:role_api_name/member/batch_remove_authorization` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 编辑角色授权用户 |
| 权限要求 | app_engine:role.member:write |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| namespace | string | 应用命名空间 |
| role_api_name | string | 角色 API 名称 |

### 请求体

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| user_ids | undefined[] | 否 | 需要删除的用户 ID 列表，使用飞书低代码平台的用户 ID <br> **示例**:  <br> **数据校验规则**:<br>最大长度: 1000最小长度: 0 |
| department_ids | undefined[] | 否 | 需要删除的部门 ID 列表，使用飞书低代码平台的部门 ID <br> **示例**:  <br> **数据校验规则**:<br>最大长度: 1000最小长度: 0 |

**请求体示例**:

```json
{
    "user_ids": [
        "1802412778084426"
    ],
    "department_ids": [
        "1802412778084426"
    ]
}
```

### 响应

**响应示例**:

```json
{
    "code": 0,
    "msg": "success",
    "data": {}
}
```

### 错误码

| HTTP状态码 | 错误码 | 描述 | 排查建议 |
| ---------- | ------ | ---- | -------- |
| 400 | 2320001 | param is invalid | 请检查输入参数 |
