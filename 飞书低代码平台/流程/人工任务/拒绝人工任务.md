## 拒绝人工任务

对于人工任务进行拒绝操作

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/apaas/v1/approval_tasks/:approval_task_id/reject` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 修改飞书低代码平台审批信息 |
| 权限要求 | app_engine:approval:write |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| approval_task_id | string | 人工任务 ID |

### 请求体

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| user_id | string | 是 | 操作人 ID <br> **示例**: 1783981209205788  |
| opinion | string | 否 | 审批意见 <br> **示例**: 拒绝  |

**请求体示例**:

```json
{
    "user_id": "1783981209205788",
    "opinion": "拒绝"
}
```

### 响应

**响应示例**:

```json
{
    "code": 0,
    "msg": "success",
    "data": {}
}
```

### 错误码

| HTTP状态码 | 错误码 | 描述 | 排查建议 |
| ---------- | ------ | ---- | -------- |
| 400 | 2320001 | param is invalid | 请检查输入参数 |
