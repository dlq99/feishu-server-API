## 获取活跃会议

获取一个预约的当前活跃会议。

⚠️ 
 只能获取归属于自己的预约的活跃会议（一个预约最多有一个正在进行中的会议）

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/vc/v1/reserves/:reserve_id/get_active_meeting` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token, user_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 获取会议预约信息 <br> 获取用户 user ID |
| 权限要求 | vc:reserve:readonly |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| reserve_id | string | 预约ID（预约的唯一标识） |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| with_participants | boolean | 否 | 是否需要参会人列表，默认为false |
| user_id_type | string | 否 | 用户 ID 类型 |
