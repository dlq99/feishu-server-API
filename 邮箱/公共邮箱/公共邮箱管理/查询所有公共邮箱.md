## 查询所有公共邮箱

分页批量获取公共邮箱列表。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/mail/v1/public_mailboxes` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token, user_access_token |
| 支持的应用类型 | custom |
| 权限要求 | 查询公共邮箱 <br> 获取用户 user ID <br> 查看公共邮箱数据驻留地 <br> 查询、创建、修改公共邮箱 |
| 权限要求 | mail:public_mailbox <br> mail:public_mailbox:readonly |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| page_token | string | 否 | 分页标记，第一次请求不填，表示从头开始遍历；分页查询结果还有更多项时会同时返回新的 page_token，下次遍历可采用该 page_token 获取查询结果 |
| page_size | integer | 否 | 分页大小 |
### 响应

**响应示例**:

```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "has_more": true,
        "page_token": "xxx",
        "items": [
            {
                "public_mailbox_id": "xxxxxxxxxxxxxxx",
                "email": "test_public_mailbox@xxx.xx",
                "name": "test public mailbox",
                "geo": "cn"
            }
        ]
    }
}
```

### 错误码

| HTTP状态码 | 错误码 | 描述 | 排查建议 |
| ---------- | ------ | ---- | -------- |
| 503 | 1235003 | Service unavailable | 请稍后重试 |
