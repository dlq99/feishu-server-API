## 获取评估模板配置

获取评估模板配置信息，包括模版名称、执行角色、填写项类型等

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/performance/v2/review_templates/query` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 管理绩效数据 <br> 查看绩效数据 <br> 获取评估配置信息 |
| 权限要求 | performance:performance <br> performance:performance:readonly <br> performance:review_template:read |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| page_token | string | 否 | 分页标记，第一次请求不填，表示从头开始遍历；分页查询结果还有更多项时会同时返回新的 page_token，下次遍历可采用该 page_token 获取查询结果 |
| page_size | integer | 否 | 分页大小 |
### 请求体

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| review_template_ids | undefined[] | 否 | 评估模板 ID 列表，获取指定评估模板的配置数据。如果不传则返回所有 <br> **示例**:  <br> **数据校验规则**:<br>最大长度: 50最小长度: 0 |

**请求体示例**:

```json
{
    "review_template_ids": [
        "7343513161666707459"
    ]
}
```
