## 直接创建待入职

使用指定数据创建一个待入职人员。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/corehr/v2/pre_hires` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom |
| 权限要求 | 更新核心人事信息 <br> 读写待入职人员信息 |
| 权限要求 | corehr:corehr <br> corehr:pre_hire:write |

### 请求体

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| basic_info | object | 是 | 个人信息 <br> **示例**:   |

| education_info | education_info[] | 否 | 教育经历 <br> **示例**:   |
