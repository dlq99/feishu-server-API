## 下载文件

根据ID下载文件

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/corehr/v1/files/:id` |
| HTTP Method | GET |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 读写飞书人事的附件信息 |
| 权限要求 | corehr:file:write |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| id | string | 上传文件ID |

### 响应

### 错误码

| HTTP状态码 | 错误码 | 描述 | 排查建议 |
| ---------- | ------ | ---- | -------- |
| 400 | 1160001 | param is invalid | 请检查file_id |
