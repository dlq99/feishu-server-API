## 查询成本分摊

查询成本分摊

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/corehr/v2/cost_allocations/batch_query` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom  isv |
| 权限要求 | 成本分摊写权限 <br> 获取用户 user ID <br> 成本分摊读权限 |
| 权限要求 | corehr:cost_allocation:read <br> corehr:cost_allocation:write |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| user_id_type | string | 否 | 用户 ID 类型 |
### 请求体

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| employment_ids | undefined[] | 是 | 员工ID列表<br>-可以调用[【搜索员工信息】](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/corehr-v2/employee/search)接口，获取指定员工的 employment_id。 <br> **示例**:  <br> **数据校验规则**:<br>最大长度: 100最小长度: 1 |

**请求体示例**:

```json
{
    "employment_ids": [
        "4692446793125560154"
    ]
}
```
