## 删除地点

删除地点信息


❌ 
 删除后无法恢复， 并且在系统中无法搜索到对应地点信息，请谨慎操作。

💡 
 - 删除对象时请确认有无在职员工、异动单据、待入职单据关联此对象，如有会导致删除失败。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/corehr/v1/locations/:location_id` |
| HTTP Method | DELETE |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom |
| 权限要求 | 更新核心人事信息 <br> 更新地点信息 |
| 权限要求 | corehr:corehr <br> corehr:locations:write |

### 路径参数

| 参数名 | 类型 |  描述 |
| ------ | ---- | ---- |
| location_id | string | 需要删除的地点 ID。ID获取方式：
- 调用[【创建地点】](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/reference/corehr-v1/location/create)[【批量分页查询地点】](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/reference/corehr-v1/location/list)等接口可以返回地点ID |

### 响应

```json
}
```
