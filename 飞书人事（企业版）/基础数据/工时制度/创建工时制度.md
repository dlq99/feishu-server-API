## 创建工时制度

创建工时制度。

## 请求

| 基本 | |
| --- | --- |
| HTTP URL | `https://open.feishu.cn/open-apis/corehr/v1/working_hours_types` |
| HTTP Method | POST |
| 支持的访问令牌 | tenant_access_token |
| 支持的应用类型 | custom |
| 权限要求 | 更新预置数据信息 <br> 更新核心人事信息 |
| 权限要求 | corehr:common_data.preset_data:write <br> corehr:corehr |

### 查询参数

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| client_token | string | 否 | 根据client_token是否一致来判断是否为同一请求 |
### 请求体

| 参数名 | 类型 | 必填 | 描述 |
| ------ | ---- | ---- | ---- |
| code | string | 否 | 编码 <br> **示例**: 1  |
| name | i18n[] | 是 | 名称 <br> **示例**:   |
| country_region_id_list | undefined[] | 否 | 国家/地区 ID 列表<br>- 可通过[查询国家/地区信息](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/corehr-v2/basic_info-country_region/search)获取 <br> **示例**: 创建成功的工时制度信息  |
| default_for_job | boolean | 是 | 职务默认值 <br> **示例**: true  |
| active | boolean | 是 | 是否启用 <br> **示例**: true  |
| custom_fields | object_field_data[] | 否 | 自定义字段<br>- 具体支持的对象请参考[自定义字段说明](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/reference/corehr-v1/custom-fields-guide) <br> **示例**:   |

**类型额外说明**:

- **i18n** 类型说明:

  基本属性:

   lang: string <br> value: string <br>
- **object_field_data** 类型说明:

  基本属性:

   field_name: string <br> value: string <br>


**请求体示例**:

```json
{
    "code": "1",
    "name": [
        {
            "lang": "zh-CN",
            "value": "张三"
        }
    ],
    "country_region_id_list": [
        "6890452208593356295"
    ],
    "default_for_job": true,
    "active": true,
    "custom_fields": [
        {
            "field_name": "name",
            "value": "\"Sandy\""
        }
    ]
}
```
