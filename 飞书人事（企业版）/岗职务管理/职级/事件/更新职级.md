# 【事件】更新职级

飞书人事中「职级信息被更新」时将触发此事件。{使用示例}(url=/api/tools/api_explore/api_explore_config?project=corehr&version=v2&resource=job_level&event=updated)

<md-alert type="error">

</md-alert>


<md-alert type="warn">
- 【查询租户职级信息】、【查询单个职级信息】、[【通过职级 ID 批量查询职级信息
】](/ssl:ttdoc/uAjLw4CM/ukTMukTMukTM/corehr-v2/job_level/batch_get)数据查询存在一定延迟(2s以内)，不建议收到事件后立即查询以上接口。
- 以下字段变更会收到事件：```name（职级名称)、active（职级启用状态）、description（职级描述）、code（编码）、job_grade（职等）、level_order（职级数值）、external_id（外键 ID）```
- 以下字段变更不会收到事件：```wk_id（ID）、wk_created_at（创建时间）、wk_created_by（创建人）、wk_updated_at（更新时间）、wk_updated_by（更新人）、wk_deleted_at（删除时间）```
</md-alert>


<md-alert type="tip">

</md-alert>


## 前提条件
你需要在应用中配置事件订阅，这样才可以在事件触发时接收到事件数据。了解事件订阅可参见事件订阅概述。

package main

import (
	"context"
	"fmt"

	larkcore "github.com/larksuite/oapi-sdk-go/v3/core"
	larkevent "github.com/larksuite/oapi-sdk-go/v3/event"
	"github.com/larksuite/oapi-sdk-go/v3/event/dispatcher"
	"github.com/larksuite/oapi-sdk-go/v3/service/corehr/v2"

)

// SDK 使用说明 SDK user guide：https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/server-side-sdk/golang-sdk-guide/preparations
func main() {
	// 注册事件 Register event
	eventHandler := dispatcher.NewEventDispatcher("", "").
		OnP2JobLevelUpdatedV2(func(ctx context.Context, event *larkcorehr.P2JobLevelUpdatedV2) error {
			fmt.Printf("[ OnP2JobLevelUpdatedV2 access ], data: %s\n", larkcore.Prettify(event))
			return nil
		})

	// 构建 client Build client
	cli := larkws.NewClient("YOUR_APP_ID", "YOUR_APP_SECRET",
		larkws.WithEventHandler(eventHandler),
		larkws.WithLogLevel(larkcore.LogLevelDebug),
	)

	// 建立长连接 Establish persistent connection
	err := cli.Start(context.Background())

	if err != nil {
		panic(err)
	}
}
